# 📥 Cách chạy project

## 1️⃣ Clone project

```bash
git clone https://github.com/azmedia2006/Cong-Nghe-Phat-Trien-Ung-Dung-Bai-3.git
```

## 2️⃣ Vào thư mục project

```bash
cd Cong-Nghe-Phat-Trien-Ung-Dung-Bai-3
```

## 3️⃣ Cài thư viện (nếu có)

NodeJS:

```bash
npm install
```

Python:

```bash
pip install -r requirements.txt
```

## 4️⃣ Chạy project

NodeJS:

```bash
npm start
```

hoặc

```bash
node server.js
```

Python:

```bash
python main.py
```
